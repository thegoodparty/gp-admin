# GP Admin Web

Next.js 15 frontend application for GP Admin.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3500](http://localhost:3500) with your browser to see the result.

## Features

- Next.js 15 with App Router
- TypeScript
- Tailwind CSS
- ESLint

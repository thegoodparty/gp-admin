# GP Admin API

NestJS backend application for GP Admin.

## Getting Started

```bash
# development
npm run dev

# production mode
npm run build
npm run start:prod
```

The API will be available at [http://localhost:3501](http://localhost:3501)

## API Endpoints

- `GET /` - Hello message
- `GET /health` - Health check endpoint

## Testing

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```

## Features

- NestJS 10
- TypeScript
- Jest for testing
- ESLint & Prettier
- CORS enabled
